About

Frets On Fire X

FoFiX is a rhythm game inspired by many of the popular commercial games. It is a fork of Frets On Fire.

Development has been pretty slow over the years but we are still here!

Projects

Here is a list of our projects:

  • FoFiX: the main game
  • fretwork: a shared library with Frets On Fire (with Cython extentions)
  • fretwork-wheels: automation to create wheels
  • fofix.github.io: this website
  • PyPitch: a C++-extension to analyse audio streams for pitch
  • VideoPlayer: a C-extension in Python to read OGG video files.
